Find all solutions (if any) to each of the following systems of linear equations.
  1. \begin{align*} x + y + 2z \amp = 8\\ 3x - y + z \amp = 0\\ -x + 3y + 4z \amp = -4 \end{align*}
  2. \begin{align*} -2x + 3y + 3z \amp = -9\\ 3x - 4y + z \amp = 5\\ -5x + 7y + 2z \amp = -14 \end{align*}
  3. \begin{align*} x + y - z \amp = 10\\ -x + 4y + 5z \amp = -5\\ x + 6y + 3z \amp = 15 \end{align*}
  4. \begin{align*} x + 2y - z \amp = 2\\ 2x + 5y - 3z \amp = 1\\ x + 4y - 3z \amp = 3 \end{align*}
  5. \begin{align*} 5x + y \amp = 2\\ 3x - y - 2z \amp = 1\\ x + y - z \amp = 5 \end{align*}
  6. \begin{align*} 3x - 2y + z \amp = -2\\ x - y + 3z \amp = 5\\ -x + y + z \amp = -1 \end{align*}
  7. \begin{align*} x + y + z \amp = 2\\ x + z \amp = 1\\ 2x + 5y + 2z \amp = 7 \end{align*}
  8. \begin{align*} x + 2y - 4z \amp = 10\\ 2x - y + 2z \amp = 5\\ x + y - 2z \amp = 7 \end{align*}
