Exercise 1.

(Source: [4], Chapter 5, Exercise 5.1.2)

In each case, determine if \(\vec{x}\) is in \(\span(\vec{y}, \vec{z})\text{.}\) If it is, write \(\vec{x}\) as a linear combination of \(\vec{y}\) and \(\vec{z}\text{.}\) If not, explain why not.

  1. \(\vec{x} = \begin{bmatrix}2\\-1\\0\\1\end{bmatrix}, \vec{y} = \begin{bmatrix}1\\0\\0\\1\end{bmatrix}, \vec{z} = \begin{bmatrix}0\\1\\0\\1\end{bmatrix}\text{.}\)

  2. \(\vec{x} = \begin{bmatrix}1 \\ 2 \\ 15 \\ 11\end{bmatrix}, \vec{y} = \begin{bmatrix}2\\-1\\0\\2\end{bmatrix}, \vec{z} = \begin{bmatrix}1\\-1\\-3\\1\end{bmatrix}\text{.}\)

  3. \(\vec{x} = \begin{bmatrix}8\\3\\-13\\20\end{bmatrix}, \vec{y} = \begin{bmatrix}2\\1\\-3\\5\end{bmatrix}, \vec{z} = \begin{bmatrix}-1\\0\\2\\-3\end{bmatrix}\text{.}\)

  4. \(\vec{x} = \begin{bmatrix}2\\5\\8\\3\end{bmatrix}, \vec{y} = \begin{bmatrix}2\\-1\\0\\5\end{bmatrix}, \vec{z} = \begin{bmatrix}-1\\2\\2\\-3\end{bmatrix}\text{.}\)
