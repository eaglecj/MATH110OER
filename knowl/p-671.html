in-context =======

In this example the vector \(\begin{bmatrix}3\\2\end{bmatrix}\) was redundant, in that removing it did not change the vectors we could get as linear combinations. The collection \(S\) can be described as either \(\span\left(\begin{bmatrix}1\\1\end{bmatrix}, \begin{bmatrix}1\\-1\end{bmatrix}, \begin{bmatrix}3\\2\end{bmatrix}\right)\) or as \(\span\left(\begin{bmatrix}1\\1\end{bmatrix}, \begin{bmatrix}1\\-1\end{bmatrix}\right)\text{,}\) but the latter description is more parsimonious. Moreover, we cannot remove any more vectors from our list and still have the span be \(S\text{,}\) because the span of a single vector in \(\mathbb{R}^2\) is a line, while we already know that \(S = \mathbb{R}^2\text{.}\) We therefore have found a collection of vectors with no redundancies whose span is \(S\text{.}\)
