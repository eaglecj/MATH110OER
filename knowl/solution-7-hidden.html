Denote by \(X\) the barycentre of the hexagon. By construction,
\begin{equation*} \vect{XA} = -\vect{XD},\; \vect{XB} = -\mbf{e},\; \vect{XC} = -\vect{XF}. \end{equation*}
Now, by definition \(\vect{AB} = \vect{OB}-\vect{OA}\text{,}\) where \(O\) denotes the origin. If we add and substract \(\vect{OX}\) on the right hand-side, we do not change the equality, so
\begin{equation*} \vect{AB} = \vect{OB}-\vect{OA} + \vect{OX} - \vect{OX} = (\vect{OB} - \vect{OX}) - (\vect{OA} - \vect{OX}) = \vect{XB} - \vect{XA}. \end{equation*}
For convenience, let us denote \(\vect{XA}, \ldots, \vect{XF}\) by \(\mbf{a}, \ldots, \mbf{f}\text{.}\) We can now rewrite the left-hand side of the equation in question as follows:
\begin{align*} \amp \overrightarrow{AB} + \overrightarrow{AC} + \overrightarrow{AD} + \overrightarrow{AE}+ \overrightarrow{AF} \\ \amp\quad = (\mbf{b}-\mbf{a}) + (\mbf{c}-\mbf{a}) + (\mbf{d}-\mbf{a}) + (\mbf{e}-\mbf{a}) + (\mbf{f}-\mbf{a}) \amp\amp\\ \amp\quad = (\mbf{b}+\mbf{d}) + (\mbf{c}+\mbf{d}) + (\mbf{d}+\mbf{d}) + (\mbf{e}+\mbf{d}) + (\mbf{f}+\mbf{d}) \amp\amp \text{ since } \mbf{a} = -\mbf{d}\\ \amp\quad = \mbf{b} + \mbf{c} + \mbf{e} + \mbf{f} + 6 \mbf{d} \amp\amp \end{align*}
Since \(\mbf{b}=-\mbf{e}\) and \(\mbf{c}=-\mbf{f}\text{,}\) we conclude:
\begin{equation*} \overrightarrow{AB} + \overrightarrow{AC} + \overrightarrow{AD} + \overrightarrow{AE}+ \overrightarrow{AF} = 6 \mbf{d} . \end{equation*}
Now, since \(\mbf{a}=-\mbf{d}\text{,}\) we can write \(2\mbf{d}=\mbf{d} - \mbf{a}\text{,}\) so that
\begin{equation*} \overrightarrow{AB} + \overrightarrow{AC} + \overrightarrow{AD} + \overrightarrow{AE}+ \overrightarrow{AF} = 3(\mbf{d} - \mbf{a}) = 3\vect{AD}, \end{equation*}
as claimed.
