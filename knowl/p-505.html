In each of the following, find (if possible) conditions on \(a,b,\) and \(c\) such that the system has one solution, no solution or infinitely many solutions.
  1. \begin{align*} 3x + y - z \amp = a\\ x - y + 2z \amp = b\\ 5x + 3y - 4z \amp = c \end{align*}
  2. \begin{align*} -x + 3y + 2z \amp = -8\\ x + z \amp = 2\\ 3x + 3y + az \amp = b \end{align*}
  3. \begin{align*} x + ay \amp = 0\\ y + bz \amp = 0\\ cx + z \amp = 0 \end{align*}
  4. \begin{align*} 3x - y + 2z \amp = 3\\ x + y - z \amp = 2\\ 2x - 2y + 3z \amp = b \end{align*}
  5. \begin{align*} x + ay - z \amp = 1\\ -x + (a-2)y + z \amp = -1\\ 2x + 2y + (a-2)z \amp = 1 \end{align*}
