There exists matrices \(A\) and \(B\) such that \(det(A+B) \ne det(A) + det(B)\) and so the statement of the exercise is false. We can show this witha counterexample. Let \(A \) be the identity matrix, \(A = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix}\) and let \(B = \begin{bmatrix} -1 \amp 0 \\ 0 \amp -1\text{,}\) then
\begin{equation*} det(A+B) = det\left(\begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} + \begin{bmatrix} -1 \amp 0 \\ 0 \amp -1 \end{bmatrix}right) = det(\begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} \ne det(A) + det(B) = 1 + 1 =2 \end{equation*}
