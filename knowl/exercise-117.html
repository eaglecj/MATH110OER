Use determinants to find which real values of c make each of the following matrices invertible.

  1. \(\begin{bmatrix} 1 \amp 0 \amp 3 \\ 3 \amp -4 \amp c \\ 2 \amp 5 \amp 8 \end{bmatrix} \) Answer. Solution.

  2. \(\begin{bmatrix} 0 \amp c \amp -c \\ -1 \amp 2 \amp 1 \\ c \amp -c \amp c \end{bmatrix} \) Answer. Solution.

  3. \(\begin{bmatrix} c \amp 1 \amp 0 \\ 0 \amp 2 \amp c \\ -1 \amp c \amp 5 \end{bmatrix} \) Answer. Solution.

  4. \(\begin{bmatrix} 4 \amp c \amp 3 \\ c \amp 2 \amp c \\ 5 \amp c \amp 4 \end{bmatrix}\) Answer. Solution.

  5. \(\begin{bmatrix} 1 \amp 2 \amp -1 \\ 0 \amp -1 \amp c \\ 2 \amp c \amp 1 \end{bmatrix} \) Answer. Solution.

  6. \(\begin{bmatrix} 1 \amp c \amp -1 \\ c \amp 1 \amp 1 \\ 0 \amp 1 \amp c \end{bmatrix} \) Answer. Solution.
