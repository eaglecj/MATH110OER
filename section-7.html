1.

(Source: [4], Chapter 2, Exercise 2.1.5)

Find \(A \) in terms of \(B \) if:
  1. \begin{equation*} A + B = 3A + 2B \end{equation*}
  2. \begin{equation*} 2A - B = 5(A + 2B) \end{equation*}
Hint
Theorem 2.1.1 in Section 2.1 of [4] might help.
Answer
  1. \begin{equation*} A= -\frac{1}{2} B \end{equation*}
  2. \begin{equation*} A = -\frac{11}{3}B \end{equation*}
Solution
Because of the properties listed in Theorem 2.1.1 of [4], we can manipulate both sides of the equations just like we would if \(A\) and \(B\) were numbers:
  1. \begin{align*} A + B = 3A + 2B \iff \amp (1-3)A = (2-1)B \\ \iff \amp -2A = B \\ \iff \amp A= -\frac{1}{2} B. \end{align*}
  2. \begin{align*} 2A - B = 5(A + 2B) \iff \amp 2A - B = 5A + 10B\\ \iff \amp -3A = 11B\\ \iff \amp A = -\frac{11}{3}B. \end{align*}

2.

(Source: [4], Chapter 2, Exercise 2.1.6)

If \(X, Y, A, \) and \(B \) are matrices of the same size, solve the following systems of equations to obtain \(X \) and \(Y \) in terms of \(A \) and \(B \text{.}\)
  1. \begin{align*} 5X + 3Y \amp = A\\ 2X + Y \amp = B \end{align*}
  2. \begin{align*} 4X + 3Y \amp = A\\ 5X + 4Y \amp = B \end{align*}
Hint
It might help to look back into Theorem 2.1.1 of [4].
Answer
  1. Answer
    \(X= -A + 3 B\text{,}\) \(Y=-5B+2A\text{.}\)
  2. Answer
    \(X= 4A - 3 B\text{,}\) \(Y=4B-5A\)
Solution
Because of the properties listed in Theorem 2.1.1 of [4], we can manipulate both sides of the equations just like we would if \(A, B\) were numbers and \(X, Y\) were variables (however, we cannot simply divide by a matrix!).
  1. Solution
    The system can also be written as
    \begin{equation*} \matr{cc|c}{ 5 \amp 3 \amp A \\ 2 \amp 1 \amp B}, \end{equation*}
    where it is important that we remember that \(A,B\) are not numbers, so we cannot divide by them. We row-reduce:
    \begin{align*} \matr{cc|c}{5 \amp 3 \amp A \\ 2 \amp 1 \amp B} \underset{5R_{2}}{\overset{2R_{1}}{\longrightarrow}} \amp \matr{cc|c}{10 \amp 6 \amp 2A \\ 10 \amp 5 \amp 5B} \overset{-R_{1}+R_{2}}{\longrightarrow} \matr{cc|c}{10 \amp 6 \amp 2A \\ 0 \amp -1 \amp 5B-2A}\\ \overset{6R_{2}+R_{1}}{\longrightarrow} \amp \matr{cc|c}{10 \amp 0 \amp 2A + 6 (5B-2A) \\ 0 \amp -1 \amp 5B-2A}\\ \amp= \matr{cc|c}{10 \amp 0 \amp -10A + 30 B \\ 0 \amp -1 \amp 5B-2A}. \end{align*}
    We conclude that \(X= \frac{1}{10} (-10A + 30 B) = -A + 3 B\) and \(Y=-5B+2A\text{.}\)
  2. Solution
    The system can also be written as
    \begin{equation*} \matr{cc|c}{ 4 \amp 3 \amp A \\ 5 \amp 4 \amp B}, \end{equation*}
    where it is important that we remember that \(A,B\) are not numbers, so we cannot divide by them. We row-reduce:
    \begin{align*} \matr{cc|c}{ 4 \amp 3 \amp A \\ 5 \amp 4 \amp B} \overset{5R_{1}}{\underset{4R_{2}}{\longrightarrow}} \amp\matr{cc|c}{ 20 \amp 15 \amp 5A \\ 20 \amp 16 \amp 4B} \overset{-R_{1}+R_{2}}{\longrightarrow} \matr{cc|c}{ 20 \amp 15 \amp 5A \\ 0 \amp 1 \amp 4B-5A}\\ \overset{-15R_{2}+R_{1}}{\longrightarrow} \amp\matr{cc|c}{ 20 \amp 0 \amp 5A-15(4B-5A) \\ 0 \amp 1 \amp 4B-5A}\\ \amp=\matr{cc|c}{ 20 \amp 0 \amp 80A-60B \\ 0 \amp 1 \amp 4B-5A} \end{align*}
    We conclude that \(X= \frac{1}{20} (80A-60B) = 4A - 3 B\) and \(Y=4B-5A\text{.}\)

3.

(Source: [3], Chapter 9, Exercise 9.2.7)

Determine whether each of the following sets of matrices is linearly indpendent. If it is linearly independent, write one matrix as a linear combination of the other matrices in the set.
  1. \begin{equation*} \left\{ \begin{bmatrix} 1 \amp 2 \\ 0 \amp 1 \end{bmatrix},\begin{bmatrix} -7 \amp 2 \\ -2\amp -3 \end{bmatrix},\begin{bmatrix} 4 \amp 0 \\ 1 \amp 2 \end{bmatrix}\right\} \end{equation*}
  2. \begin{equation*} \left\{ \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix},\begin{bmatrix} 0 \amp 1 \\ 0\amp 1 \end{bmatrix},\begin{bmatrix} 1 \amp 0 \\ 1 \amp 0 \end{bmatrix},\begin{bmatrix} 0 \amp 0 \\ 1 \amp 1 \end{bmatrix}\right\} \end{equation*}
Hint 1
Matrices \(A_{1}, \ldots, A_{m}\) of the same size are called linearly dependent if there exist numbers \(c_{1},\ldots, c_{m}\text{,}\) not all of which are zero, so that
\begin{equation*} c_{1}A_{1}+c_{2}A_{2}+\ldots + c_{m}A_{m} = 0. \end{equation*}
Otherwise, they are called linearly independent.
Hint 2
Another way of phrasing linear independence is: Matrices \(A_{1}, \ldots, A_{m}\) of the same size are linearly independent if the only solution \((c_{1}, \ldots, c_{m})\) to the system of linear equations
\begin{equation*} x_{1}A_{1}+x_{2}A_{2}+\ldots + x_{m}A_{m} = 0 \end{equation*}
is the trivial solution, \((0,\ldots, 0).\)
Answer
  1. Answer
    The matrices are linearly independent.
  2. Answer
    The matrices are linearly independent.
Solution
  1. Solution
    We are wondering whether there exist numbers \(x,y,z\text{,}\) not all zero, such that
    \begin{equation*} x \begin{bmatrix} 1 \amp 2 \\ 0 \amp 1 \end{bmatrix} + y\begin{bmatrix} -7 \amp 2 \\ -2\amp -3 \end{bmatrix} + z\begin{bmatrix} 4 \amp 0 \\ 1 \amp 2 \end{bmatrix}\right\} = \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix}. \end{equation*}
    Note that an equality of matrices means componentwise equality, so for example, the equality of the \((1,1)\)-entry says:
    \begin{equation*} x - 7 y + 4 z = 0. \end{equation*}
    Proceeding in this way, we arrive at a system of 4 linear equations with augmented matrix that we can row-reduce:
    \begin{align*} \matr{ccc|c} { 1 \amp -7 \amp 4 \amp 0 \\ 2 \amp 2 \amp 0 \amp 0 \\ 0 \amp -2 \amp 1 \amp 0 \\ 1 \amp -2 \amp 2 \amp 0 } \overset{-2R_{1}+R_{2}}{\underset{-R_{1}+R_{4}}{\longrightarrow}} \amp\matr{ccc|c} { 1 \amp -7 \amp 4 \amp 0 \\ 0 \amp 16 \amp -8 \amp 0 \\ 0 \amp -2 \amp 1 \amp 0 \\ 0 \amp 5 \amp -2 \amp 0 } \overset{8R_{3}+R_{2}}{\underset{2R_{4}}{\longrightarrow}} \matr{ccc|c} { 1 \amp -7 \amp 4 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp -2 \amp 1 \amp 0 \\ 0 \amp 10 \amp -4 \amp 0 }\\ \overset{5R_{3}+R_{4}}{\longrightarrow} \amp \matr{ccc|c} { 1 \amp -7 \amp 4 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp -2 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 } \end{align*}
    At this point, we can already see that further row reduction will yield that the only solution is \((x,y,z)=(0,0,0)\text{.}\) Therefore, the matrices are linearly independent.
  2. Solution
    We are wondering whether there exist numbers \(w,x,y,z\text{,}\) not all zero, such that
    \begin{equation*} w\begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} +x\begin{bmatrix} 0 \amp 1 \\ 0\amp 1 \end{bmatrix} +y\begin{bmatrix} 1 \amp 0 \\ 1 \amp 0 \end{bmatrix} +z\begin{bmatrix} 0 \amp 0 \\ 1 \amp 1 \end{bmatrix} = \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} . \end{equation*}
    Just as in the last part, we could turn this into an augmented matrix of size \(4\times 5\) and row-reduce, but since the entries of these matrices are easier, we will do it by hand:

    Equality of the \((1,2)\)-entries says that \(x=0\text{.}\)

    Equality of the \((1,1)\)-entries says that \(w+y=0\text{,}\) i.e. \(w=-y\text{.}\)

    Equality of the \((2,1)\)-entries says that \(y+z=0\text{,}\) i.e. \(z=-y=w\text{.}\)

    Lastly, equality of the \((2,2)\)-entries says that \(w+x+z=0\text{,}\) which means that \(w=-z\) since \(x=0\text{.}\)

    Since \(-z=w=z\) also, we conclude that \(z=0\) and so also \(0=z=-y=w\text{.}\) All in all, the only solution is \((w,x,y,z)=(0,0,0,0)\text{,}\) so the matrices are linearly independent.

4.

(Source: [5], LISS.C20

Let
\begin{equation*} S = \left\{ \begin{bmatrix} 1 \amp 2 \\ 2 \amp 1 \end{bmatrix}, \begin{bmatrix} 2 \amp 1 \\ -1 \amp 2 \end{bmatrix}, \begin{bmatrix} 0 \amp 1 \\ 1 \amp 2 \end{bmatrix} \right\} \end{equation*}
  1. Determine if \(S \) spans the set of all \(2 \times 2 \) matrices.
  2. Determine if \(S \) is linearly independent.
Hint
See and for the definition of linear independence.
Answer
  1. Answer
    No, \(S\) does not span the set of all \(2\times 2\) matrices.
  2. Answer
    Yes, the set \(S\) is linearly independent.
Solution
  1. Solution
    The set \(S \) spans the set of all \(2 \times 2 \) matrices if and only if, for any \(2 \times 2 \) matrix \(A\text{,}\) there exist numbers \(x_{1}, x_{2}, x_{3}\) so that
    \begin{equation*} x_{1} \begin{bmatrix} 1 \amp 2 \\ 2 \amp 1 \end{bmatrix} + x_{2}\begin{bmatrix} 2 \amp 1 \\ -1 \amp 2 \end{bmatrix} + x_{3} \begin{bmatrix} 0 \amp 1 \\ 1 \amp 2 \end{bmatrix} = A . \end{equation*}
    If we write \(A = \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\text{,}\) then this is equivalent to saying that there exists a solution to the following system of linear equations:
    \begin{align*} x_{1} + 2x_{2} \amp = a\\ 2 x_{1} + x_{2} + x_{3} \amp= b\\ 2x_{1} - x_{2} + x_{3} \amp = c\\ x_{1} + 2x_{2} + 2x_{3} \amp = d \end{align*}
    Let us transfer this into an augmented matrix and then row-reduce:
    \begin{align*} \amp\matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 2 \amp 1 \amp 1 \amp b \\ 2 \amp -1 \amp 1 \amp c \\ 1 \amp 2 \amp 2 \amp d } \overset{-R_{1}+R_{4}}{\underset{-2R_{1}+R_{3}, -2R_{1}+R_{2}}{\longrightarrow}} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -3 \amp 1 \amp b-2a \\ 0 \amp -5 \amp 1 \amp c-2a \\ 0 \amp 0 \amp 2 \amp d - a }\\ \amp\overset{\frac{1}{2}R_{4}}{\underset{-R_{4}'+R_{3}, -R_{4}'+R_{2}}{\longrightarrow}} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -3 \amp 0 \amp b-2a -\frac{1}{2}(d - a) \\ 0 \amp -5 \amp 0 \amp c-2a - \frac{1}{2}(d - a) \\ 0 \amp 0 \amp 1 \amp \frac{1}{2}(d - a) }\\ \amp\overset{5R_{2}}{\underset{-3R_{3}}{\longrightarrow}} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -15 \amp 0 \amp 5b-\frac{15}{2}a -\frac{5}{2}d \\ 0 \amp 15 \amp 0 \amp -3c+ \frac{9}{2}a + \frac{3}{2}d \\ 0 \amp 0 \amp 1 \amp \frac{1}{2}(d - a) }\\ \amp\overset{R_{2}+R_{3}}{\longrightarrow} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -15 \amp 0 \amp 5b-\frac{15}{2}a -\frac{5}{2}d \\ 0 \amp 0 \amp 0 \amp (-3c+ \frac{9}{2}a + \frac{3}{2}d) + (5b-\frac{15}{2}a -\frac{5}{2}d) \\ 0 \amp 0 \amp 1 \amp \frac{1}{2}(d - a) }. \end{align*}
    We see from the third row that the system is inconsistent if
    \begin{equation*} -3c -3 a - d + 5b = \left(-3c+ \frac{9}{2}a + \frac{3}{2}d) + (5b-\frac{15}{2}a -\frac{5}{2}d\right) \neq 0. \end{equation*}
    Since \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}\) is an example of a choice of numbers which makes the system inconsistent, we conclude that, no, \(S\) does not span the set of all \(2\times 2\) matrices.
  2. Solution
    We need to figure out wheter
    \begin{equation*} x_{1} \begin{bmatrix} 1 \amp 2 \\ 2 \amp 1 \end{bmatrix} + x_{2}\begin{bmatrix} 2 \amp 1 \\ -1 \amp 2 \end{bmatrix} + x_{3} \begin{bmatrix} 0 \amp 1 \\ 1 \amp 2 \end{bmatrix} = \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} \end{equation*}
    has a non-zero solution. We reuse our computation from Solution 3.2.4.a.1 for the case \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix}\text{:}\)
    \begin{align*} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 2 \amp 1 \amp 1 \amp b \\ 2 \amp -1 \amp 1 \amp c \\ 1 \amp 2 \amp 2 \amp d } \amp\overset{\text{row reduction}}{\longrightarrow} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp 0 \\ 0 \amp -15 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 }\\ \amp\overset{2R_{2}+15R_{1}}{\longrightarrow} \matr{ccc|c} { 15 \amp 0 \amp 0 \amp 0 \\ 0 \amp -15 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 }. \end{align*}
    From this, we see that the only solution to the system of equations
    \begin{equation*} x_{1} \begin{bmatrix} 1 \amp 2 \\ 2 \amp 1 \end{bmatrix} + x_{2}\begin{bmatrix} 2 \amp 1 \\ -1 \amp 2 \end{bmatrix} + x_{3} \begin{bmatrix} 0 \amp 1 \\ 1 \amp 2 \end{bmatrix} = \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} \end{equation*}
    is \((x_{1},x_{2},x_{3})=(0,0,0)\text{.}\) Thus, the set \(S\) is linearly independent.

5.

(Source: [1], Chapter 2, Exercise 2.1.14)

Let \(A = \begin{bmatrix} 1 \amp 2 \\ 3 \amp 4 \end{bmatrix}\) and \(B = \begin{bmatrix} 1 \amp 2 \\ 3 \amp k \end{bmatrix}\text{.}\) Is it possible to find \(k\) such that \(AB = BA\text{?}\) If so, what should \(k \) equal?
Hint
Compute both \(AB\) and \(BA\) and compare the answer.
Answer
Yes, \(k=4\text{.}\)
Solution
We compute
\begin{equation*} AB = \begin{bmatrix} 7 \amp 2+2k \\ 15 \amp 6+4k \end{bmatrix} \text{ and } BA = \begin{bmatrix} 7 \amp 10 \\ 3+3k \amp 6+4k \end{bmatrix}. \end{equation*}
For these to be equal, we need them to be equal entrywise. The \((1,1)\)- and the \((2,2)\)-entry coincide for every choice of \(k\text{,}\) and for the other two entries we need
\begin{equation*} 2 + 2k = 10, \text{ i.e. } k = 4, \text{ and also }15 = 3+3k, \text{ i.e. } k= 4. \end{equation*}
We see that \(k=4\) makes \(AB=BA\text{.}\)

6.

(Source: [1], Chapter 2, Exercise 2.1.15)

Let \(A = \begin{bmatrix} 1 \amp 2 \\ 3 \amp 4 \end{bmatrix}\) and \(B = \begin{bmatrix} 1 \amp 2 \\ 1 \amp k \end{bmatrix}\text{.}\) Is it possible to find \(k\) such that \(AB = BA\text{?}\) If so, what should \(k \) equal?
Hint
Compute both \(AB\) and \(BA\) and compare the answer.
Answer
No.
Solution
We compute
\begin{equation*} AB = \begin{bmatrix} 3 \amp 2+2k \\ 7 \amp 6+4k \end{bmatrix} \text{ and } BA = \begin{bmatrix} 7 \amp 10 \\ 1+3k \amp 6+4k \end{bmatrix}. \end{equation*}
For these to be equal, we need them to be equal entrywise, but no matter what \(k\) we choose, the \((1,1)\)-entries will never coincide (\(3\neq 7\)). Thus, the answer is no.

7.

(Source: [4], Chapter 2, Exercise 2.1.16)

Let \(A \) and \(B \) be square matrices of the same size.
  1. If \(A \) and \(B \) are symmetric, then so is \(A - B \text{.}\)
  2. If \(A\) is symmetric, then so is \(kA \) for any scalar \(k \text{.}\)
Hint
See Hint 3.1.5.1 for the defintion of symmetric.
Solution
  1. Solution
    Let \(A=[a_{ij}]\) and \(B=[b_{ij}]\text{,}\) so that \(A-B=[a_{ij}-b_{ij}]\text{.}\) Since \(A=A^T\text{,}\) we have \(a_{ij} = a_{ji}\) for all choices of \(i,j\text{,}\) and similarly \(b_{ij} = b_{ji}\text{.}\) Thus,
    \begin{align*} (A-B)^{T} \amp= [a_{ij}-b_{ij}]^{T} = [a_{ji}-b_{ji}] \\ \amp= [a_{ij}-b_{ij}] = [a_{ij}]-[b_{ij}] = A-B, \end{align*}
    so \(A-B\) is symmetric.
  2. Solution
    We need to show that \((kA)^{T} = k A\) to conclude that \(kA\) is symmetric. Since equality of matrices is entrywise equality, we will consider their \((i,j)\)-entries:

    Let \(A=[a_{ij}]\text{.}\) By definition of scalar multiplication, \(kA=[ka_{ij}]\text{.}\) By definition of the transpose, the \((i,j)\)-entry of \((kA)^{T}\) is therefore \(ka_{ji}\) (where we swapped the roles of \(i\) and \(j\)).

    Since \(A\) is symmetric, we know that \(a_{ij}=a_{ji}\) for any choice of \(i,j\text{,}\) so in particular \(ka_{ij}=ka_{ji}\) also. This means that the \((i,j)\)-entry of \(kA\) (namely \(ka_{ij}\)) coincides with the \((i,j)\)-entry of \((kA)^{T}\) (namely \(ka_{ji}\)), which shows that \(k A = (kA)^{T} \) since they are equal entrywise.

8.

(Source: [1], Chapter 2, Exercise 2.1.29)

A square matrix is called skew-symmetric if \(A^T = -A \text{.}\) Show that the main diagonal of a skew-symmetric matrix consists of only zeros.
Hint
If \(A=[a_{ij}]\text{,}\) then the main diagonal consists of every entry of the matrix which is of the form \(a_{ii} \text{.}\)
Solution
Write \(A=[a_{ij}]\text{.}\) By assumption,
\begin{equation*} [a_{ji}] = A^{T} = -A = [-a_{ij}], \end{equation*}
so that \(a_{ji}=-a_{ij} \) for all choices of \(i,j\text{.}\) In particular, if \(i=j\text{,}\) then \(a_{ii}=-a_{ii} \text{,}\) so \(a_{ii}=0\text{.}\)

9.

(Source: [4], Chapter 2, Exercise 2.1.17)

Show that \(A + A^T \) is symmetric for any square matrix \(A \text{.}\)
Hint
See Hint 3.1.5.1 for the definition of symmetric.
Solution
Recall first that \((A+B)^T = A^T + B^T\) for any matrices \(A, B\) of the same size. Now, since \(A\) is square, \(A\) and \(A^T\) are of the same size, so we may use this fact for \(B=A^T\text{:}\)
\begin{align*} (A+A^T)^T \amp= A^T + (A^T)^T\\ \amp = A^T + A,\qquad\text{ since } (A^T)^=A\\ \amp = A + A^T,\qquad\text{ since addition is commutative.} \end{align*}
This shows that \(A+A^T\) is its own transpose, i.e. symmetric.
