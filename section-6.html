1.

(Source: [4], Chapter 2, Exercise 2.1.3)

Let \(A = \begin{bmatrix} 2 \amp 1 \\ 0 \amp -1 \end{bmatrix}, B = \begin{bmatrix} 3 \amp -1 \amp 2\\ 0 \amp 1 \amp 4 \end{bmatrix}, C = \begin{bmatrix} 3 \amp -1 \\ 2 \amp 0 \end{bmatrix}, D = \begin{bmatrix} 1 \amp 3 \\ -1 \amp 0 \\ 1 \amp 4 \end{bmatrix},\) and \(E = \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \end{bmatrix}.\) Compute the following (where possible).
  1. \(3A - 2B \)
  2. \(5C \)
  3. \(3E^T \)
  4. - \(B+D \)
  5. \(4A^T - 3C \)
  6. \((A+C)^T \)
  7. \(2B - 3E \)
  8. \(A - D \)
  9. \((B- 2E)^T \)
Hint 1

(Source: [4], Chapter 2, Definition 2.1)

Recall: If \(A\) and \(B\) are matrices of the same size, their sum\(A+B\) is the matrix formed by adding corresponding entries.
Hint 2

(Source: [4], Chapter 2, Definition 2.2)

Recall: If \(A\) is any matrix and \(k\) is any number, the scalar multiple\(kA\) is the matrix obtained from \(A\) by multiplying each entry of \(A\) by \(k\) .
Hint 3

(Source: [4], Chapter 2, Definition 2.3)

Recall: If \(A\) is a \(m\times n\) matrix, the transpose of \(A\text{,}\) written \(A^T\text{,}\) is the \(n\times m\) matrix whose rows are just the columns of \(A\) in the same order.
Answer
  1. Answer
    Does not exist.
  2. Answer
    \begin{equation*} 5C = \begin{bmatrix} 15 \amp -5 \\ 10 \amp 0 \end{bmatrix}\text{.} \end{equation*}
  3. Answer
    \begin{equation*} 3E^T = \begin{bmatrix} 3 \amp 0 \\ 0 \amp 3 \\ 3 \amp 0 \end{bmatrix}. \end{equation*}
  4. Answer
    Does not exist.
  5. Answer
    \begin{align*} 4A^T - 3C \amp = \begin{bmatrix} -1 \amp 3 \\ -2 \amp -4 \end{bmatrix}. \end{align*}
  6. Answer
    \begin{align*} (A+C)^T \amp = \begin{bmatrix} 5 \amp 2 \\ 0 \amp -1 \end{bmatrix}. \end{align*}
  7. Answer
    \begin{align*} 2B - 3E \amp = \begin{bmatrix} 3 \amp -2 \amp 1\\ 0 \amp -1 \amp 8 \end{bmatrix}. \end{align*}
  8. Answer
    Does not exist.
  9. Answer
    \begin{align*} (B- 2E)^T \amp = \begin{bmatrix} 1\amp 0 \\ -1 \amp -1 \\ 0 \amp 4 \end{bmatrix}. \end{align*}
Solution
  1. Solution
    We cannot add these matrices as they are not of the same size (\(3A\) is a \(2\times 2\) matrix, while \(2B\) is a \(2\times 3\) matrix).
  2. Solution
    \begin{equation*} 5C = 5\begin{bmatrix} 3 \amp -1 \\ 2 \amp 0 \end{bmatrix} = \begin{bmatrix} 15 \amp -5 \\ 10 \amp 0 \end{bmatrix}\text{.} \end{equation*}
  3. Solution
    \begin{equation*} 3E^T = 3 \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \end{bmatrix}^{T} =3 \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \\ 1 \amp 0 \end{bmatrix} = \begin{bmatrix} 3 \amp 0 \\ 0 \amp 3 \\ 3 \amp 0 \end{bmatrix}. \end{equation*}
  4. Solution
    We cannot add these matrices as they are not of the same size (\(-B\) is a \(2\times 3\) matrix, while \(D\) is a \(3\times 2\) matrix).
  5. Solution
    \begin{align*} 4A^T - 3C \amp = 4 \begin{bmatrix} 2 \amp 1 \\ 0 \amp -1 \end{bmatrix}^T - 3\begin{bmatrix} 3 \amp -1 \\ 2 \amp 0 \end{bmatrix}\\ \amp = 4 \begin{bmatrix} 2 \amp 0 \\ 1 \amp -1 \end{bmatrix} - \begin{bmatrix} 9 \amp -3 \\ 6 \amp 0 \end{bmatrix}\\ \amp = \begin{bmatrix} 8 - 9 \amp 0 - (-3) \\ 4 - 6 \amp -4 - 0 \end{bmatrix}\\ \amp = \begin{bmatrix} -1 \amp 3 \\ -2 \amp -4 \end{bmatrix}. \end{align*}
  6. Solution
    \begin{align*} (A+C)^T \amp = \left( \begin{bmatrix} 2 \amp 1 \\ 0 \amp -1 \end{bmatrix} + \begin{bmatrix} 3 \amp -1 \\ 2 \amp 0 \end{bmatrix}\right)^T \\ \amp = \begin{bmatrix} 5 \amp 0 \\ 2 \amp -1 \end{bmatrix}^T \\ \amp = \begin{bmatrix} 5 \amp 2 \\ 0 \amp -1 \end{bmatrix}. \end{align*}
  7. Solution
    \begin{align*} 2B - 3E \amp = 2 \begin{bmatrix} 3 \amp -1 \amp 2\\ 0 \amp 1 \amp 4 \end{bmatrix} - 3 \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \end{bmatrix}\\ \amp = \begin{bmatrix} 6 \amp -2 \amp 4\\ 0 \amp 2 \amp 8 \end{bmatrix} - \begin{bmatrix} 3 \amp 0 \amp 3 \\ 0 \amp 3 \amp 0 \end{bmatrix}\\ \amp = \begin{bmatrix} 3 \amp -2 \amp 1\\ 0 \amp -1 \amp 8 \end{bmatrix}. \end{align*}
  8. Solution
    We cannot add these matrices as they are not of the same size (\(A\) is a \(2\times 2\) matrix, while \(-D\) is a is a \(3\times 2\) matrix).
  9. Solution
    \begin{align*} (B- 2E)^T \amp = \left( \begin{bmatrix} 3 \amp -1 \amp 2\\ 0 \amp 1 \amp 4 \end{bmatrix} - 2 \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \end{bmatrix}\right)^{T}\\ \amp = \begin{bmatrix} 3 - 2 \amp -1 - 0 \amp 2 - 2\\ 0 - 0 \amp 1 - 2 \amp 4 - 0 \end{bmatrix}^{T}\\ \amp = \begin{bmatrix} 1 \amp -1 \amp 0\\ 0 \amp -1 \amp 4 \end{bmatrix}^{T}\\ \amp = \begin{bmatrix} 1\amp 0 \\ -1 \amp -1 \\ 0 \amp 4 \end{bmatrix}. \end{align*}

2.

(Source: [3], Chapter 4, Exercise 4.4.6)

Consider the matrices: \(A = \begin{bmatrix} 1 \amp 2 \amp 3\\ 2 \amp 1 \amp 7 \end{bmatrix}, B = \begin{bmatrix} 3 \amp -1 \amp 2 \\ -3 \amp 2 \amp 1 \end{bmatrix}, C = \begin{bmatrix} 1 \amp 2 \\ 3 \amp 1 \end{bmatrix}, D = \begin{bmatrix} -1 \amp 2 \\ 2 \amp -3 \end{bmatrix}, \) and \(E = \begin{bmatrix} 2 \\ 3 \end{bmatrix}.\) Find the following if possible. If it is not possible, explain why.
  1. \(-3A \)
  2. \(3B - A \)
  3. \(AC \)
  4. \(CB \)
  5. \(AE \)
  6. \(EA \)
Hint

(Source: [3], Chapter 4, Definition 4.21)

Recall: Let \(A=[a_{i j}]\) be an \(m\times n\)-matrix, and let \(B=[b_{j k}]\) be an \(n\times p\)-matrix. Then their product is the \(m\times p\)-matrix \(AB= [c_{ik}]\) whose \((i,k)\)-entry is defined by
\begin{equation*} c_{ik} = a_{i1}b_{1k}+a_{i2}b_{2k}+...+a_{in}b_{nk}. \end{equation*}
Answer
  1. Answer
    \begin{align*} -3A \amp=\begin{bmatrix} -3 \amp -6 \amp -9\\ -6 \amp -3 \amp -21 \end{bmatrix}. \end{align*}
  2. Answer
    \begin{align*} 3B - A \amp = \begin{bmatrix} 8 \amp -5 \amp 3 \\ -11 \amp 5 \amp -4 \end{bmatrix}. \end{align*}
  3. Answer
    Does not exist.
  4. Answer
    \begin{align*} CB = \begin{bmatrix} -3 \amp 3 \amp 4 \\ 6 \amp -1 \amp 7 \end{bmatrix}. \end{align*}
  5. Answer
    Does not exist.
  6. Answer
    Does not exist.
Solution
  1. Solution
    \begin{align*} -3A \amp= -3\begin{bmatrix} 1 \amp 2 \amp 3\\ 2 \amp 1 \amp 7 \end{bmatrix} =\begin{bmatrix} -3 \amp -6 \amp -9\\ -6 \amp -3 \amp -21 \end{bmatrix}. \end{align*}
  2. Solution
    \begin{align*} 3B - A \amp =3\begin{bmatrix} 3 \amp -1 \amp 2 \\ -3 \amp 2 \amp 1 \end{bmatrix} - \begin{bmatrix} 1 \amp 2 \amp 3\\ 2 \amp 1 \amp 7 \end{bmatrix}\\ \amp = \begin{bmatrix} 3\cdot 3 -1 \amp 3\cdot (-1) - 2 \amp 3\cdot 2 - 3 \\ 3\cdot (-3) -2 \amp 3\cdot 2 -1 \amp 3\cdot 1 -7 \end{bmatrix}\\ \amp = \begin{bmatrix} 8 \amp -5 \amp 3 \\ -11 \amp 5 \amp -4 \end{bmatrix}. \end{align*}
  3. Solution
    We cannot multiply these matrices as their sizes do not match up: the number of columns in \(A\) does not coincide with the number of rows in \(C\text{.}\)
  4. Solution
    \begin{align*} CB \amp = \begin{bmatrix} 1 \amp 2 \\ 3 \amp 1 \end{bmatrix}\begin{bmatrix} 3 \amp -1 \amp 2 \\ -3 \amp 2 \amp 1 \end{bmatrix}\\ \amp = \begin{bmatrix} 1 \cdot 3 + 2\cdot (-3) \amp 1\cdot (-1) + 2\cdot 2 \amp 1\cdot 2 + 2\cdot 1 \\ 3 \cdot 3 + 1\cdot (-3) \amp 3\cdot (-1) + 1\cdot 2 \amp 3\cdot 2 + 1\cdot 1 \end{bmatrix}\\ \amp = \begin{bmatrix} -3 \amp 3 \amp 4 \\ 6 \amp -1 \amp 7 \end{bmatrix}. \end{align*}
  5. Solution
    We cannot multiply these matrices as their sizes do not match up: the number of columns in \(A\) does not coincide with the number of rows in \(E\text{.}\)
  6. Solution
    We cannot multiply these matrices as their sizes do not match up: the number of columns in \(E\) does not coincide with the number of rows in \(A\text{.}\)

3.

(Source: [3], Chapter 1, Exercise 1.3.6)

Write the following system of linear equations as an augmented matrix (of the form \(A\mathbf{x} = \mathbf{b}\)).
\begin{align*} x - 3z - 2y \amp = 5\\ 6 -x \amp = 4+y - z\\ 2x + 3 \amp = x+3y \end{align*}
Hint
First simplify and rearrange the equations so that ''like'' variables are lined up in columns. Write the variables in the order \(x,y,z\text{.}\)
Answer
\begin{align*} \matr{ccc|c}{ 1 \amp -2 \amp - 3 \amp 5\\ -1 \amp -1 \amp 1 \amp -2\\ 1 \amp -3 \amp 0 \amp -3 } \end{align*}
Solution
We sort by ''like'' variables:
\begin{align*} x - 2y - 3z \amp = 5\\ -x - y + z \amp = -2\\ x -3y \amp = -3 \end{align*}
Now we can easily write it as augmented matrix:
\begin{align*} \matr{ccc|c}{ 1 \amp -2 \amp - 3 \amp 5\\ -1 \amp -1 \amp 1 \amp -2\\ 1 \amp -3 \amp 0 \amp -3 }. \end{align*}

4.

(Source: [4], Chapter 2, Exercise 2.3.29)

Let \(A\) and \(B\) be \(n \times n\) matrices for which the system of equations \(A\mathbf{x} = \mathbf{0}\) and \(B\mathbf{x} = \mathbf{0}\) each only have the trivial solution \(\mathbf{x} = \mathbf{0}\text{.}\) Show that the system \(AB \mathbf{x} = \mathbf{0}\) has only the trivial solution.
Hint 1
Assume that \(\mathbf{y}\) is a solution to the system \(AB \mathbf{x} = \mathbf{0}\text{.}\) Use the assumptions to conclude that we must have \(\mathbf{y}=\mathbf{0}\text{.}\)
Hint 2
Remember that \(AB \mathbf{x} = A(B \mathbf{x}) =(AB) \mathbf{x}\text{.}\)
Solution
Take any vector \(\mathbf{y}\) with \(n\) components such that \(AB \mathbf{y} = \mathbf{0}\text{.}\) This means that \(B \mathbf{y}\) is a solution to the system of equations \(A\mathbf{x} = \mathbf{0}\) which, by assumption, only has the trivial solution. In other words, \(B \mathbf{y} = \mathbf{0}\text{.}\) This similarly means that \(\mathbf{y}\) is a solution to the system of equations \(B\mathbf{x} = \mathbf{0}\) which, again by assumption, only has the trivial solution. Thus, \(\mathbf{y}=\mathbf{0}\text{.}\) We have seen that the system \(AB \mathbf{x} = \mathbf{0}\) only has the trivial solution.

5.

(Source: [4], Chapter 2, Exercise 2.3.27)

In each case either show the statement is true or give an example showing that it is false. (We assume that the matrices are of the correct size so that all matrix products which we consider make sense.)
  1. If \(A^2 = I\text{,}\) then \(A = I\text{.}\)
  2. If \(AB = A\text{,}\) then \(B = I\text{.}\)
  3. If \(A \) is square, then \((A^T)^3 = (A^3)^T\text{.}\)
  4. If \(A\) is symmetric, then \(I + A\) is symmetric.
  5. If \(AB = AC \) and \(A \ne 0\text{,}\) then \(B = C \text{.}\)
  6. If \(A \ne 0\text{,}\) then \(A^2 \ne 0\text{.}\)
  7. If \(A \) commutes with \(A+B \) then \(A \) commutes with \(B \text{.}\)
Hint 1

See Hint 3.1.1.3 for the definition of the transpose and Hint 3.1.2.1 for the definition of matrix multiplication.

Look back into [4], Chapter 2, Theorem 2.1.1, for some properties of matrix operation.

A square matrix \(A\) is called symmetric if \(A=A^T\text{.}\)

Two square matrices \(A,B\) of the same size are said to commute if \(AB=BA\text{.}\)

Hint 2
All counterexamples can be chosen to be of size \(2\times 2\text{.}\)
Answer
  1. False.
  2. False.
  3. True.
  4. True.
  5. False.
  6. False.
  7. True.
Solution
  1. Solution
    False: Take \(A =\begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix}\neq I\text{,}\) which satisfies \(A^2 = I\text{.}\)
  2. Solution
    False: Take \(A=B=\begin{bmatrix} 1 \amp 0 \\ 0 \amp 0\end{bmatrix}\neq I\text{,}\) which satisfy \(AB = A\text{.}\)
  3. Solution
    True: We will prove that \(B^T A^T=(AB)^T\text{,}\) which will imply our claim because
    \begin{equation*} (A^T)^3 = (A^T)(A^T)(A^T) = A^T(AA)^T = ((AA)A)^T = (A^3)^T. \end{equation*}
    For this proof, let us write \(A_{i,j}\) for the \((i,j)\)-entry of \(A\text{.}\) Thus, we know from Hint 3.1.2.1 that the \((i,k)\)-entry of \(AB\) is given by:
    \begin{equation*} (AB)_{i,k} = A_{i,1}B_{1,k}+A_{i,2}B_{2,k}+...+A_{i,n}B_{n,k}. \end{equation*}
    Recall further from Hint 3.1.1.3 that \((B^T)_{k,j}=B_{j k}\) and \((A^{T})_{j,i}=A_{i j}\text{,}\) so that \(B^T A^T\) has \((k,i)\)-entry
    \begin{align*} (B^T A^T)_{k,i} \amp= (B^T)_{k,1}(A^T)_{1,i}+(B^T)_{k,2}(A^T)_{2,i}+...+(B^T)_{k,n}(A^T)_{n,i}\\ \amp = B{1,k}A_{i,1}+B_{2,k}A_{i,2}+...+B_{n,k}A_{i,n}\\ \amp = (AB)_{i,k}. \end{align*}
    Again by Hint 3.1.1.3, \((i,k)\)-entry of \(AB\) is the same as the \((k,i)\)-entry of \((AB)^T\text{,}\) so that
    \begin{equation*} (B^T A^T)_{k,i}= (AB)_{i,k} = ((AB)^T)_{k,i} . \end{equation*}
    We have shown that the \((k,i)\)-entry of \(B^T A^T\) and of \((AB)^T\) are identical. This means that \(B^T A^T=(AB)^T\text{.}\)
  4. Solution
    True:\(A\) is symmetric means \(A=A^T\text{.}\) Thus, \((I + A)^{T} = I^{T}+A^{T} = I+A\text{,}\) where the first equality follows from Theorem 2.1.2(4) in [4]. Thus, \(I + A\) is symmetric as well.
  5. Solution
    False: Let
    \begin{equation*} A=\begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}, B=\begin{bmatrix} 1 \amp 0 \\ 1 \amp 1 \end{bmatrix}, C=\begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix}. \end{equation*}
    Then \(A \ne 0\) and
    \begin{equation*} AB = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix} = A = AC, \end{equation*}
    but \(B \neq C \text{.}\)
  6. Solution
    False: Let \(A=\begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \end{bmatrix}\text{.}\) Then \(A^2 = 0\text{,}\) even though \(A\neq 0\text{.}\)
  7. Solution
    True: We compute
    \begin{equation*} A^2 + AB = A(A+B) \overset{(*)}{=} (A+B)A = A^2 + BA, \end{equation*}
    where \((*)\) is by assumption that \(A\) commutes with \(A+B\text{.}\) If we substract \(A^{2}\) from both the left- and right-most side of the equation, we see that \(AB=BA\text{,}\) i.e. \(A\) commutes with \(B \text{.}\)

6.

(Adapted from: [4], Chapter 2, Exercise 2.3.27)

In each case either show the statement is true or give an example showing that it is false. (We assume that the matrices are of the correct size so that all matrix products which we consider make sense.)
  1. If \(B\) has a row of zeros, so also does \(AB\) for all \(A\text{.}\)
  2. If \(A \) has a row of zeros, so also does \(AB \) for all \(B\text{.}\)
  3. If \(B\) has a column of zeros, so also does \(AB \) for all \(A\text{.}\)
  4. If \(AB \) has a column of zeros for some \(A\neq 0\text{,}\) then so also does \(B\text{.}\)
  5. If \(AB \) has a row of zeros for some \(A\neq 0\text{,}\) so also does \(B \text{.}\)
Answer
  1. False.
  2. True.
  3. True.
  4. False.
  5. False.
Solution
  1. Solution
    False: Let
    \begin{equation*} A=\begin{bmatrix} 1 \amp 1 \\ 1 \amp 1 \end{bmatrix} \text{ and } B=\begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}. \end{equation*}
    Then \(AB = \begin{bmatrix} 1 \amp 0 \\ 1 \amp 0 \end{bmatrix}\) does not have a zero row.
  2. Solution
    True: Let us write \(A= [ a_{ij}]\) and \(B= [ b_{jk}]\text{.}\) By assumption, there exists a zero row in \(A\text{,}\) say row \(i_{0}\text{.}\) In other words, \(a_{i_{0}j}=0\) for all \(j\text{.}\) By definition of \(AB= [c_{ik}]\) (see Hint 3.1.2.1), its \((i,k)\)-entry is defined by
    \begin{equation*} c_{ik} = a_{i1}b_{1k}+a_{i2}b_{2k}+...+a_{in}b_{nk}. \end{equation*}
    Since \(a_{i_{0}j}=0\) for all \(j\text{,}\) we see that \(c_{i_{0}k}=0\) for all \(k\text{.}\) In other words, row \(i_{0}\) in \(AB\) is zero.
  3. Solution
    True: Let us write \(A= [ a_{ij}]\) and \(B= [ b_{jk}]\text{.}\) By assumption, there exists a zero column in \(B\text{,}\) say column \(k_{0}\text{.}\) In other words, \(b_{jk_{0}}=0\) for all \(j\text{.}\) By definition of \(AB= [c_{ik}]\) (see Hint 3.1.2.1), its \((i,k)\)-entry is defined by
    \begin{equation*} c_{ik} = a_{i1}b_{1k}+a_{i2}b_{2k}+...+a_{in}b_{nk}. \end{equation*}
    Since \(b_{jk_{0}}=0\) for all \(j\text{,}\) we see that \(c_{ik_{0}}=0\) for all \(i\text{.}\) In other words, column \(k_{0}\) in \(AB\) is zero.
  4. Solution
    False: Take
    \begin{equation*} A=\begin{bmatrix} 1 \amp -2 \\ 2 \amp -4 \end{bmatrix} \text{ and } B=\begin{bmatrix} 2 \amp 4 \\ 1 \amp 2 \end{bmatrix}, \end{equation*}
    so that
    \begin{equation*} AB =\begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} . \end{equation*}
  5. Solution
    False: Take
    \begin{equation*} A=\begin{bmatrix} 1 \amp -2 \\ 2 \amp -4 \end{bmatrix} \text{ and } B=\begin{bmatrix} 2 \amp 4 \\ 1 \amp 2 \end{bmatrix}, \end{equation*}
    so that
    \begin{equation*} AB =\begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} . \end{equation*}
